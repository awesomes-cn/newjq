# newjq
A jQuery plugin collection website


## Requirements

* Ruby 2.2.0 +
* Rails 4.2.0
* Mysql 5.0+
* ImageMagick 6.5+

## Install
Clone the project

```bash
$ git clone git@github.com:awesomes-cn/newjq.git
```
Create a mysql database and migrate 
```bash
$ rake db:migrate

```
Init some data

```bash
$ rake dbdata

```
Run app
```bash
$ rails s -b 0.0.0.0

```
